| Dataset | Source | Description |
|---|---|---|
| MIMIC-IV | https://physionet.org/content/mimiciv/ | Clinical prediction — ICU patient data |
| Kaggle Tabular Datasets | https://www.kaggle.com/datasets | Classification and regression benchmarks |
| PhysioNet Challenge Data | https://physionet.org/about/challenge/ | Annual clinical prediction competitions |
| UCI ML Repository | https://archive.ics.uci.edu/ | Standard classification benchmarks |

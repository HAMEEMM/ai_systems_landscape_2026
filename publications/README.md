## Overview

This directory contains 20 research papers — one **unified taxonomy paper** (#00) covering all 19 AI system types, plus **19 individual deep-dive papers** (one per AI type). Each paper follows a dual-submission strategy: first to a **preprint server** for immediate visibility, then to a **peer-reviewed journal** for formal publication.

## Submission Workflow

For each paper, follow this order:

1. **Write the manuscript** — LaTeX (`acmart` for ACM, `article` for arXiv, journal-specific for others)
2. **Collect data** — Run scripts in `data/`, download public datasets
3. **Submit preprint** — Upload to the assigned preprint server (see table above)
4. **Record the preprint DOI** — Add to the journal submission cover letter
5. **Format for peer review** — Convert to the target journal's template
6. **Submit to journal** — Include the preprint DOI in the cover letter
7. **Track status** — Update checklists in the `{server}-pre-print/` and `{journal}-peer-review/` README files

## Notes

- **Diverse portfolio:** All 20 preprint servers and all 20 peer-review venues are unique — each paper targets a different platform, building a broad research presence across CS, medical, engineering, and interdisciplinary communities.

- **Backup venues** are listed in each paper's peer-review README in case of rejection.
- **LaTeX templates** vary by venue — see each paper's peer-review README for format requirements.

## Why aiXiv?

- Purpose-built preprint server for AI research — accepts papers from human, robot, and AI scientists
- AI-assisted peer review with transparent process
- Free, open-access platform with fast publication turnaround
- Active research community (see [arXiv:2508.15126](https://arxiv.org/abs/2508.15126) for platform details)
- GitHub: [aixiv-org](https://github.com/aixiv-org) | Discord community

## Submission Steps

1. **Create an aiXiv account** at https://aixiv.science/sign-up
2. **Prepare files:**
   - `aixiv-paper.pdf` (compiled PDF)
   - `references.bib` (optional supplementary)
   - Any figure files (TikZ diagrams are inline, so none needed)
3. **Upload** via https://aixiv.science/submit
   - Upload the compiled PDF
   - Add metadata: title, abstract, authors, keywords, license
4. **Review the submission** — check metadata and PDF rendering
5. **Submit** — paper goes through AI-assisted peer review
6. **Record the aiXiv DOI/ID** for the peer-review submission

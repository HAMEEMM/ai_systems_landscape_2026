# Data — Paper #6: Conversational AI

## Public Datasets for Conversational AI Research

This folder will contain scripts and metadata for publicly available datasets used in the paper.

### Suggested Datasets

| Dataset | Source | Description |
|---|---|---|
| MultiWOZ | https://github.com/budzianowski/multiwoz | Multi-domain task-oriented dialogue |
| DailyDialog | https://aclanthology.org/I17-1099/ | Human daily conversation dataset |
| PersonaChat | https://github.com/facebookresearch/ParlAI | Persona-conditioned dialogue |
| DSTC (Dialogue System Technology Challenge) | https://dstc.community/ | Yearly challenge datasets |

### Data Collection

Scripts for data collection and processing will be added here as the paper develops.

## Submission Steps

1. **Create an arXiv account** at https://arxiv.org/user/register (if not already done)
2. **Get endorsement** — new authors in `cs.AI` need endorsement from an existing arXiv author
3. **Prepare files:**
   - `arxiv-paper.tex` (already created — arXiv-compatible article format)
   - `references.bib`
   - Any figure files (TikZ diagrams are inline, so none needed)
4. **Upload** via https://arxiv.org/submit
   - Select category: `cs.AI`, cross-list: `cs.LG`
   - Upload `.tex` + `.bib` as a zip or individual files
   - arXiv compiles the PDF on their servers using TeX Live
5. **Review the compiled PDF** — check formatting, figures, references
6. **Submit** — paper goes live within 1–2 business days
7. **Record the arXiv ID** (e.g., `arXiv:2603.XXXXX`) for the peer-review submission

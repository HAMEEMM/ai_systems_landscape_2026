# Data — Paper #8: Explainable AI (XAI)

## Public Datasets for XAI Research

This folder will contain scripts and metadata for publicly available datasets used in the paper.

### Suggested Datasets

| Dataset | Source | Description |
|---|---|---|
| COMPAS Recidivism | https://github.com/propublica/compas-analysis | Fairness and explainability benchmark |
| German Credit | https://archive.ics.uci.edu/dataset/144/statlog+german+credit+data | Credit scoring — XAI evaluation |
| MIMIC-III | https://physionet.org/content/mimiciii/ | Clinical decision explanation |
| ImageNet (for saliency maps) | https://www.image-net.org/ | Visual explanation benchmarks |

# Data — Paper #11: Multimodal Perception AI

## Public Datasets for Multimodal AI Research

This folder will contain scripts and metadata for publicly available datasets used in the paper.

### Suggested Datasets

| Dataset | Source | Description |
|---|---|---|
| COCO Captions | https://cocodataset.org/ | Image captioning — vision + language |
| VQA v2.0 | https://visualqa.org/ | Visual question answering |
| AudioSet | https://research.google.com/audioset/ | Audio event classification |
| MIMIC-CXR | https://physionet.org/content/mimic-cxr/ | Chest X-ray + radiology reports |

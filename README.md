## How to Use

1. **Open `index.html`** in any browser (or host on GitHub Pages)
2. **Browse the master hub** — taxonomy, classification, comparison, market data, glossary, infographics
3. **Click "Explore Deep Dive →"** on any of the 19 AI system types
4. **Navigate between types** using the prev/next buttons at the bottom of each page
5. **Return to the hub** via the "← AI Systems Landscape" button (top-left on every page)

## What Each Interactive Chart Covers

Every HTML page is a comprehensive, self-contained reference with interactive elements, animated infographics, and full narrative content:

1. **Overview & Definition** — what the AI type is, with a comparison table against all 18 other AI types
2. **How It Works** — interactive pipeline diagrams and process breakdowns
3. **Core Architectures** — key technical approaches with strengths and trade-offs
4. **The 8-Layer Stack** — expandable layers from hardware to application
5. **Sub-Types** — detailed taxonomy within the AI type
6. **Deep-Dive Sections** — domain-specific technical explorations (3 per type)
7. **Platforms, Frameworks & Tools** — interactive tables of the current ecosystem
8. **Evaluation & Benchmarks** — live Chart.js visualisations of performance data
9. **Industry Use Cases** — real-world applications by sector
10. **Market & Adoption Data** — interactive growth charts and investment data
11. **Risks, Limitations & Safety** — categorised risk cards and mitigation strategies
12. **Regulation & Governance** — EU AI Act, NIST, and global frameworks
13. **Glossary** — searchable terminology (50–80 terms per page)
14. **Infographics** — Animation overview and tech-stack GIFs
15. **Cross-References** — links to related AI system types in the landscape

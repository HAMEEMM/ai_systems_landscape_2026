# Data — Paper #12: Optimization / Operations Research AI

## Public Datasets for OR AI Research

This folder will contain scripts and metadata for publicly available datasets used in the paper.

### Suggested Datasets

| Dataset | Source | Description |
|---|---|---|
| TSPLIB | http://comopt.ifi.uni-heidelberg.de/software/TSPLIB95/ | TSP and VRP benchmark instances |
| OR-Library | http://people.brunel.ac.uk/~mastjjb/jeb/info.html | Classic OR benchmark problems |
| MIPLIB | https://miplib.zib.de/ | Mixed-integer programming benchmarks |
| Google OR-Tools Benchmarks | https://github.com/google/or-tools | Routing and scheduling benchmarks |

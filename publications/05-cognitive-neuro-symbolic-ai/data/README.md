| Dataset | Source | Description |
|---|---|---|

| bAbI Tasks | https://research.facebook.com/downloads/babi/ | Reasoning and QA tasks |
| ARC (Abstraction and Reasoning Corpus) | https://github.com/fchollet/ARC | Abstract reasoning benchmark |
| KGQA Benchmarks | Various | Knowledge graph question answering |

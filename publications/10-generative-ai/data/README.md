# Data — Paper #10: Generative AI

## Public Datasets for Generative AI Research

This folder will contain scripts and metadata for publicly available datasets used in the paper.

### Suggested Datasets

| Dataset | Source | Description |
|---|---|---|
| LAION-5B | https://laion.ai/blog/laion-5b/ | Large-scale image-text pairs for diffusion models |
| The Pile | https://pile.eleuther.ai/ | 800GB diverse text corpus for LLM training |
| HumanEval | https://github.com/openai/human-eval | Code generation benchmark |
| MMLU | https://github.com/hendrycks/test | Massive multitask language understanding |

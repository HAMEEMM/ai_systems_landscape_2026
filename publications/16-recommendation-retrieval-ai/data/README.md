# Data — Paper #16: Recommendation / Retrieval AI

## Public Datasets for Recommendation AI Research

This folder will contain scripts and metadata for publicly available datasets used in the paper.

### Suggested Datasets

| Dataset | Source | Description |
|---|---|---|
| MovieLens | https://grouplens.org/datasets/movielens/ | Movie recommendation benchmark |
| Amazon Product Reviews | https://nijianmo.github.io/amazon/index.html | E-commerce recommendation data |
| MS MARCO | https://microsoft.github.io/msmarco/ | Passage retrieval benchmark |
| BEIR | https://github.com/beir-cellar/beir | Heterogeneous retrieval benchmark |

# Data — Paper #17: Reinforcement Learning AI

## Public Datasets for RL Research

This folder will contain scripts and metadata for publicly available datasets used in the paper.

### Suggested Datasets

| Dataset | Source | Description |
|---|---|---|
| Atari 2600 (ALE) | https://github.com/mgbellemare/Arcade-Learning-Environment | Classic RL benchmark — 57 games |
| MuJoCo Benchmarks | https://github.com/google-deepmind/mujoco | Continuous control tasks |
| D4RL | https://github.com/Farama-Foundation/D4RL | Offline RL benchmark suite |
| OpenAI Gym / Gymnasium | https://gymnasium.farama.org/ | Standard RL environments |

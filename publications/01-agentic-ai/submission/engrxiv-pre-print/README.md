- Established preprint server for engineering and computer science research

- Broad engineering and CS readership
- Fast moderation turnaround (typically 1–3 days)

## Submission Steps

1. **Create an engrXiv account** at https://engrxiv.org/
2. **Prepare files:**
   - `engrxiv-paper.pdf` (compiled PDF)
   - `references.bib` (optional supplementary)
   - Any figure files (TikZ diagrams are inline, so none needed)
3. **Upload** via the engrXiv submission portal
   - Upload the compiled PDF
   - Add metadata: title, abstract, authors, keywords, license
4. **Review the submission** — check metadata and PDF rendering
5. **Submit** — paper goes through moderation review
6. **Record the engrXiv DOI** for the peer-review submission

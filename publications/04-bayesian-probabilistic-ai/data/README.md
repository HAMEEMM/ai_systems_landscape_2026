This folder will contain scripts and metadata for publicly available datasets used in the paper.

| Dataset | Source | Description |
|---|---|---|
| UCI ML Repository | https://archive.ics.uci.edu/ | Benchmark datasets for uncertainty quantification |
| Bayesian Benchmarks | https://github.com/hughsalimbeni/bayesian_benchmarks | Standard regression/classification benchmarks |
| Corrupted CIFAR-10/100 | https://zenodo.org/record/2535967 | Distribution shift evaluation |
| MIMIC-III | https://physionet.org/content/mimiciii/ | Clinical data for uncertainty in medical AI |

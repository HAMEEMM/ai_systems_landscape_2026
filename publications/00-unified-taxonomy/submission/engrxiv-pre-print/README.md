## Submission Steps

1. **Create an OSF account** at https://osf.io/register (if not already done)
2. **Prepare files:**
   - `engrxiv-paper.pdf` (compiled PDF)
   - `references.bib` (optional supplementary)
   - Any figure files (TikZ diagrams are inline, so none needed)
3. **Upload** via https://engrxiv.org/submit
   - Upload the compiled PDF
   - Add metadata: title, abstract, authors, keywords, license
4. **Review the submission** — check metadata and PDF rendering
5. **Submit** — paper goes live within 1–3 business days after basic checks
6. **Record the engrXiv DOI** (e.g., `10.31224/XXXXX`) for the peer-review submission
